1, 자동정렬
Shift + Alt +F / 전체 선택후 Ctrl + K + F 
	단축키를 누르면 자동 들여쓰기 및 스타일 포맷을 한 번에 맞출 수 있습니다.

2. 터미널 열기/닫기 : Ctrl + ` (백틱)  

3. 일치하는 텍스트 바꾸기 : Ctrl + H 
	대문자 구별, 단어완전 일치, 정규식, 하나씩 바꾸기, 모두 바꾸기

4. 주석 
	한 줄 주석 토글하기 : Ctrl + /	
	한 줄에도 적용 가능하지만 여러 줄에도 사용이 가능합니다.(블록이 아닌 한줄 주석이 여러줄에 적용됨

5. 폰트 사이즈 조절 : Ctrl + '+' / Ctrl + '-'

6. 해당 라인으로 이동하기 : Ctrl + G 
	100번째 줄로 이동할 경우 100을 입력합니다.

7. 한줄 복사 : 마우스 포인터 해당 위치 이동 후 Shift + Alt + (화살표 위, 아래)

8. 한줄 이동 : 마우스 포인터 해당 위치 이동 후 Alt + (화살표 위, 아래)

9. 한줄 삭제 : 마우스 포인터 해당 위치 이동 후 Ctrl + Shift + K / Ctrl + X

10. 에러 발생시 없는 모듈 자동으로 찾아주기(중요) : Ctrl + .(점) 

11. 블록 주석으로 묶기 : 마우스 포인터 해당 코드 부분 드래그 하고 Alt + Shift + A

12. <sup></sup>같이 <>안에 있는 단어 동시에 바꾸기 : 선택하고 Ctrl + D 후 바꿀단어 쓰기

13. 커서 여러 곳에 만들기 - Alt + 마우스클릭 후 타이핑
